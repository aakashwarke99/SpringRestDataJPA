# SpringRestDataJPA

CRUD Application using below technologies

Java 8

Spring Boot 2.3.11

mysql 8

maven 4.0.0

spring data JPA

To get all courses

Method type = GET
http://localhost:8080/courses/

To get Single course by course id

Method type = GET

http://localhost:8080/courses/{courseId}

Add new course

Method Type = POST

http://localhost:8080/courses/

Update the Course

Method Type = PUT

http://localhost:8080/courses/

Delete the Course

Method Type = DELETE

http://localhost:8080/courses/{courseId}

Sample JSON

{

"id": 2,

"description": "Java 8 Book",

"title": "dennis ritchie"
}
